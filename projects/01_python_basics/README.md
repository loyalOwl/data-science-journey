# Проект 1. Трекер прогресса

Цель — потренировать функции, числа, списки и проверку кода. Проект связан с реальными данными Кирилла: масса тела и учебные занятия.

## Задание 1

Реализуй функцию `weight_gain`, которая возвращает, сколько килограммов набрано.

Пример:

```python
weight_gain(61, 70) == 9
```

## Задание 2

Реализуй `average_gain_per_week`. Функция получает начальный вес, текущий вес и число недель. Если недель 0 или меньше, нужно выбросить `ValueError`.

## Задание 3

Реализуй `completed_rate`. Функция получает список значений `True`/`False` и возвращает процент выполненных занятий от 0 до 100. Для пустого списка верни `0.0`.

## Как работать

1. Открой `progress_tracker.py`.
2. Замени только строки с `TODO`.
3. Запусти из корня проекта:

```powershell
.\.venv\Scripts\python.exe -m unittest discover -s projects/01_python_basics/tests -v
```

Красный тест — это не провал, а точная подсказка о следующем шаге.
