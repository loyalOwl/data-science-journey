📝 КАК ДЕЛАТЬ ЗАМЕТКИ

После каждого видео достаточно записать:

• название темы;
• какую проблему она решает;
• как работает в общих чертах;
• где находится в архитектуре агента;
• что осталось непонятно.

Особенно важно отметить вопросы про:

• workflow и автономного агента;
• structured outputs и function calling;
• state, memory и context;
• RAG и обычную базу данных;
• tool, API и MCP;
• tests и evals;
• agent loop и stopping conditions;
• ответственность модели, программы и человека.

После поездки обучение не начинается заново. Вопросы из заметок станут основой следующих практических уроков.
